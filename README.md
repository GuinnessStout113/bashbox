# jackboxsh

A terminal-based, multiplayer party game inspired by Jackbox, built entirely in Bash.

## Goals
- Text-only gameplay
- Multiplayer over local network / SSH
- Command-driven input (no raw prompts)
- Fun IT-themed games

## Current Status
🚧 Early prototype — Quiplash-style game mode

## Usage (WIP)
```bash
./bin/jackboxsh host
./bin/jackboxsh join localhost
``
